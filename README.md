# 🎬 PickerHut

**PickerHut** é uma plataforma web inspirada no IMDB e Rotten Tomatoes que permite pesquisar, avaliar e guardar filmes e séries, com uma funcionalidade única: pesquisar filmes em comum entre dois atores.
